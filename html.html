Hypertext Markup Language (HTML)

HTML is the standard markup language for creating web pages. It describes the structure and content of a web document using a system of elements represented by tags. HTML is interpreted by web browsers to display formatted text, images, links, forms, tables, and multimedia. The current standard is HTML5, which added semantic elements, audio/video support, and APIs.
