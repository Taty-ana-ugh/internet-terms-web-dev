File Transfer Protocol (FTP)

File Transfer Protocol (FTP) is a standard network protocol used to transfer files between a client and a server over a TCP/IP network. FTP uses two separate channels: a command channel (port 21) for sending instructions and a data channel (port 20) for transferring file content. FTPS and SFTP are secure variants that encrypt the transmission.
