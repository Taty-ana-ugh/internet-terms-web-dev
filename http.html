Hypertext Transfer Protocol (HTTP)

HTTP is the application-level protocol used for transmitting hypermedia documents on the Web. It defines how messages are formatted and transmitted between browsers and servers. HTTPS is the secure, encrypted variant of HTTP using SSL/TLS. HTTP is stateless — each request is independent.
