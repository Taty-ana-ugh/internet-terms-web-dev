Multitier Architecture

Multitier (or n-tier) architecture is a software design pattern in which the application is divided into separate logical and/or physical tiers, each with a distinct role. The most common is the 3-tier architecture: Presentation Tier (client), Logic Tier (application server), and Data Tier (database). This separation improves scalability, maintainability, and security.
